# 🔧 Comandos Git

## Verificar status

```bash
git status
```

## Clonar repositório

```bash
git clone URL_DO_REPOSITORIO
```

## Atualizar projeto

```bash
git pull origin main
```

## Enviar alterações

```bash
git push
```

## Adicionar arquivos

```bash
git add .
```

## Criar commit

```bash
git commit -m "Mensagem do commit"
```

## Trocar de branch

```bash
git checkout nome-da-branch
```

## Ver branches

```bash
git branch
```

## Juntar branches

```bash
git merge nome-da-branch
```

## Baixar informações do GitHub

```bash
git fetch
```

## Ver diferenças no código

```bash
git diff
```

## Ver histórico de commits

```bash
git log
```
